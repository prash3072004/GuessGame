# GuessGame
Number Guessing Game (Python)
A simple Python console game where the player tries to guess a randomly generated number within a limited number of attempts.

Description:
This is a Number Guessing Game built using Python
The computer randomly selects a number between 0 and 100, and the player has 3 chances to guess it correctly.

The game also:
Handles invalid (non-integer) input gracefully
Allows the player to replay the game
Ends cleanly when the player chooses to exit

Maximum of 3 attempts per round
Replay option (Y/N)

How the Game Works
The program generates a random number between 0 and 100.
The player is prompted to guess the number.
The player has 3 attempts.
If the guess is correct - You win!
If all attempts are used - You lose, and the correct number is shown.
The player can choose to play again or exit.
